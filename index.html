Orthomosaic Mapping of Urban Development

Objective: To generate a high-resolution orthomosaic of a residential development using drone imagery.

Tools: DJI Mavic 3 Pro, Pix4D Mapper, QGIS

My Role: Operated UAV for image acquisition, processed the orthomosaic in Pix4D, and analyzed land features in QGIS.

Result: Produced a 2.5 cm/pixel resolution map used for land planning and utility mapping.

Urban Expansion Monitoring Using GEE

Objective: To detect urban sprawl in Multan between 2015–2023 using Sentinel-2 satellite imagery.

Tools: Google Earth Engine, JavaScript API

My Role: Developed a script to extract NDVI, LULC classification, and visualize change over time.

Result: Identified 23% increase in built-up area. Visuals used in planning reports.
